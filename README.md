# repositório notas
Este projeto consiste em um script Python desenvolvido utilizando os princípios de Programação Orientada a Objetos (POO) para realizar o cálculo da média de um aluno, com uma interface de terminal aprimorada.
🚀 Funcionalidades

    Estrutura baseada em classes e objetos.

    Cálculo automático da média aritmética.

    Interface visual rica via terminal (tabelas, cores ou painéis).

🛠️ Tecnologias Utilizadas

    Python 3.14

    Rich (Biblioteca para renderização de conteúdo rico no terminal)

📂 Estrutura do Código

O código utiliza uma classe principal para representar o aluno, onde os atributos (notas) e métodos (cálculo e exibição) estão encapsulados, garantindo uma organização clara
